# Установка зависимостей

В консоли проекта требуется прописать команду
`pip install -r requirements.txt`
для установки всех зависимотей из файла.

# Файл конфигурации `config.json`

- **database_path** - путь к обрабатываемому .xlsx файлу относительно файла main.py
> "database_path": "data/Insurance.xlsx"
- **start_row** - номер строки (в нумерации самой таблицы Excel) с которой начинается обрабатываемая выборка (значение "None" равнозначно началу таблицы)
> "start_row": "150"
- **end_row** - номер строки (в нумерации самой таблицы Excel) которой заканчивается обрабатываемая выборка (значение "None" равнозначно концу таблицы)
> "end_row": "700"

### Функционал в разработке

- **exact_row_num** - номер строки (в нумерации самой таблицы Excel) из которой требуется получить ошибки (значение "None" равнозначно отсутвию таковой, т.е. выводятся ошибки из всех строк)
> "exact_row_num": "500"
- **exact_col_char** - буква столбца (в нумерации самой таблицы Excel) из которого требуется получить ошибки (значение "None" равнозначно отсутвию такового, т.е. выводятся ошибки из всех столбцов)
> "exact_col_char": "AF"

> "exact_col_char": "C"
